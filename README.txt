# **Artificial Neural Networks**

This repository contains Python implementations for various Artificial Neural Networks (ANN) concepts and techniques, organized by units. Each unit includes programs that demonstrate key concepts and algorithms in ANN, such as perceptrons, multi-layer perceptrons, backpropagation, and more.

## **Repository Structure**

### **Folders**
- **UNIT-1**: Basic ANN programs covering perceptrons, matrix operations, and gradient descent.
- **UNIT-2**: Advanced perceptron models, including logic gate implementations and perceptron learning rules.
- **UNIT-3**: Training neural networks with techniques like gradient descent and optimization.
- **UNIT-4**: Implementation of multi-layer perceptrons and backpropagation algorithms.
- **UNIT-5**: Practical applications, including data preprocessing, classification, and RBF networks.

### **File**
- **`req.txt`**: Contains the required Python libraries and dependencies.

---

## **Unit Details**

### **UNIT-1**
- **Matrix Operations**: Addition, multiplication, and transposition.
- **Gradient Descent**: Implementation of gradient descent for optimization.

### **UNIT-2**
- **Logic Gates**: Implementations of AND, OR, and XOR gates using perceptrons.
- **Perceptron Training**: Training perceptrons with step activation functions.

### **UNIT-3**
- **Optimization**: Programs demonstrating gradient descent optimization techniques.
- **Neural Network Training**: Training single and multi-layer perceptrons for specific tasks.

### **UNIT-4**
- **Multi-Layer Perceptrons**: Feed-forward neural network implementation.
- **Backpropagation**: Explanation and implementation of the backpropagation algorithm.

### **UNIT-5**
- **Data Classification**: Preprocessing data and performing classification using MLP and SLP models.
- **RBF Networks**: Implementation of radial basis function networks for classification tasks.
